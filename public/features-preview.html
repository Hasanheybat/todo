Görüş xülasəsi

Bu sprint ərzində əsas fokus mobil tətbiqin beta versiyası və maliyyə hesabatları moduluna yönələcək. @Leyla Məmmədova frontend, @Rəşad Əliyev backend hissəsini aparacaq.

Müştəri tərəfindən ən çox istənilən funksiya Excel export-dur. Bu sprint-də tamamlanmalıdır.

Tapşırıq siyahısı

  • Maliyyə Excel export endpoint (backend)
  • Dashboard KPI kartları
  • Mobil app — giriş ekranı
  • Mobil app — tapşırıq siyahısı
  • E-poçt bildiriş sistemi test
  • Swagger API docs genişləndir
